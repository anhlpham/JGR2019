This is a repository for data files and MATLAB scripts for the manuscript "Anthropogenic iron and nitrogen deposition alters the carbon balance in the Southern Indian Ocean" by Anh L.D. Pham and Takamitsu Ito. 
Please be aware that this is a still ongoing project.
PI folder stores results of the Pre Industrial run
Results_Modern folder stores results of the Industrial run
PI_Fe folder stores results of the Pre-Fe run
PI_N folder stores results of the Pre-N run
WOA folder stores the World Ocean Atlas file
NPP_VGPM is matlab file storing the satellite-estimated net primary production
