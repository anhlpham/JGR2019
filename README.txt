This is a repository for data files and MATLAB scripts for the manuscript "Anthropogenic iron deposition alters the ecosystem and carbon balance of the Indian Ocean over a centennial timescale" by Anh L.D. Pham and Takamitsu Ito. 
Please be aware that this is a still ongoing project.
PI.zip file stores results of the Pre Industrial run
Industrial_100yr.zip file stores results of each year for 100 years of model running under the anthropogenic nutrient deposition
WOA folder stores the World Ocean Atlas file
